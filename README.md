# DistilBERT Sentiment Analyzer

A lightweight **sentiment analysis tool built with Hugging Face Transformers**.

This project demonstrates how to use a **distilled BERT model** to classify text sentiment (positive / negative) using Python.

The model used is:

distilbert-base-uncased-finetuned-sst-2-english

It is trained on the **Stanford Sentiment Treebank (SST-2)** dataset.

---

## Project Overview

This mini-project explores **NLP sentiment classification** using Hugging Face pipelines.

The system can:

- Analyze a single text input
- Analyze multiple texts from a CSV file
- Output sentiment labels and confidence scores

The sentiment analyzer is implemented using the Hugging Face `pipeline()` API.

---

## Features

- Fast sentiment analysis
- Supports single text input
- CSV batch processing
- Clean CLI interface
- Lightweight model (DistilBERT)
